Unit Flow Solutions Dashboard — Design Language Brief

Overall Vibe
Clean, modern SaaS dashboard with a confident, business-forward personality. Think "premium productivity tool" — not flashy, but polished. The UI is bright and airy on white cards, contrasted against a rich dark green environment.

Typography

Brand/headings: Rounded, geometric sans-serif — something like Nunito, Plus Jakarta Sans, or DM Sans. The "Arcedia" wordmark feels slightly bold and approachable.
Body/data: Clean, neutral sans — Geist, Figtree, or similar. Legible at small sizes.
Numbers/stats: Larger weight (700–800), displayed prominently. Use font-variant-numeric: tabular-nums.
No serifs anywhere. Keep it tech-forward.


Layout & Spacing

Three-column layout: narrow icon sidebar → wider message/notification panel → main content area
Cards use generous internal padding (~24px) with border-radius: 16px
Tight but breathable — data-dense without feeling cramped
Stats row at top: 3 equal-width KPI cards side by side
Charts and tables below in a 2-col then full-width pattern


UI Components & Style Details

Sidebar icons: Monochrome, rounded, minimal — active state gets a filled green circle background
KPI cards: Icon in a rounded dark square badge (top left), large bold number, small trend indicator with arrow + percentage text below
Charts: Rounded-top bar chart using green fills; bars use a diagonal hatch/stripe pattern for non-highlighted bars, solid green for highlighted. A floating tooltip label on the peak bar.
Gauge/speedometer: Semi-circular progress arc, dark-to-light green gradient segments, centered number
Tables: No heavy borders — just subtle row dividers, avatar + name in first column, colored hyperlink-style emails
Buttons/dropdowns: Pill-shaped, light gray background, small chevron. Very subtle.
Notification list: Left icon, description text, right-aligned timestamp or dollar amount


Iconography
Rounded, outlined icons — consistent stroke weight. Nothing too sharp or geometric. Feels approachable.

Personality Keywords

Trustworthy · Energetic · Clean · Data-forward · Professional-but-not-boring

The green palette is doing the heavy lifting for brand energy — everything else should stay clean and get out of the way.
