## Interaction Contract

### Property Insights Table

- **Default sort:** Revenue Exposure descending — worst-performing properties surface immediately
- **Sort behavior:** Click column header toggles asc/desc; active sort column shows `ChevronUp` or `ChevronDown` icon (12px, `text-text-secondary`) — identical to ActiveJobsTable
- **Sortable columns:** Property Name (alpha), Active Turns (numeric), Avg Turn Time (numeric, nulls last), Revenue Exposure (numeric)
- **Clickable rows:** `cursor-pointer` on `<tr>`, `hover:bg-surface` background on hover — mirrors `ClickableTurnRow` pattern
- **Click action:** Navigate to `/regional/property/[encodeURIComponent(propertyName)]`
- **Empty state:** Renders inside a Card, no table rendered (see Copywriting section)

### Avg Turn Time Chart

- **Orientation:** Vertical bars (properties on X-axis, days on Y-axis) — omit `layout` prop, use Recharts default
- **Tooltip:** Hover-only value display — `formatter={(value) => [\`${value} days\`, 'Avg Turn Time']}` — no labels on bars themselves
- **Chart height:** Fixed at 220px (`height={220}` on ResponsiveContainer)
- **Margins:** `{ top: 8, right: 16, bottom: 40, left: 0 }` — extra bottom margin for angled X-axis labels
- **X-axis ticks:** `angle={-35}`, `textAnchor="end"`, truncate at 16 characters with ellipsis if `propertyName.length > 16`
- **Y-axis ticks:** Integer values only (days), min 0, no axis line, no tick lines
- **Bar radius:** `[4, 4, 0, 0]` (rounded top corners)
- **Omit:** Properties with `avgTurnTime === null` (no completed turns) are excluded from chart data
- **Card heading:** "Avg Turn Time by Property" — `font-heading text-xl font-bold text-text-primary`

### Drill-down Page

- **Back link position:** Above all Suspense boundaries, full width, margin-bottom 8px
- **Back link style:** `inline-flex items-center gap-1 text-white/70 hover:text-white text-sm` + `ArrowLeft` icon at size=14
- **Back link label:** "Back to Dashboard"
- **Layout:** Identical to `property/page.tsx` — KPIs row, then Open Turns section, then Active Jobs section
- **Edit capabilities:** Full PM edit (status dropdown, lease-ready date, job dates/status) — role='rm' passed to all components

### Sidebar Navigation

- **New nav item:** `{ icon: Map, label: "Regional Dashboard", href: "/regional", roles: ["rm"] }`
- **Active state fix:** Change active check for "Regional Dashboard" item to `activePath.startsWith('/regional')` so drill-down sub-routes highlight the nav item correctly
- **Icon:** `Map` from lucide-react (size=18 — matches all other nav icons)
- **Position:** Above "Properties" in navItems array for RM users

### Loading States

Each of the three Suspense boundaries on the RM dashboard page has a dedicated skeleton:

| Section | Skeleton Shape |
|---------|---------------|
| KPI row | 6 skeleton cards in a 3×2 grid — reuse `PMKPISkeleton` |
| Property Insights | Card with header + 3 skeleton rows (40px height each) |
| Avg Turn Time Chart | Card with header + gray rectangle 220px tall |
